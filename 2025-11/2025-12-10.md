# 2025-12-10 TIL

## 오늘 배운 것

- 자바스크립트 복습 및 심화과정, local storage, JSDoc
- Date 객체는 사용하기에 다소 불편한 점(월이 0부터 시작 등)이 있어 date-fns, day.js 같은 라이브러리를 사용해 날짜를 더 쉽고 안전하게 다루는 게 좋다.
- NaN은 NaN == NaN도 false고, NaN === NaN도 false라는 특성이 있다.
- Local Storage
  - 저장 범위: 도메인/서브도메인
    - 브라우저마다 독립적, 도메인과 서브도메인 구분
    - 예: 크롬과 파이어폭스의 google.com 데이터는 서로 다름
    - 예: 크롬에서 example.com과 sub.example.com 데이터는 서로 다름
  - 수명: 직접 지우지 않는 한 반영구적으로 남음
  - 데이터 휘발성: 탭을 닫아도 남아 있음
  - 용량: 도메인/서브도메인당 약 5~10MB
  - 통신: 서버에 자동 전송되지 않음
- Session Storage
  - 저장 범위: 탭 + 도메인/서브도메인
    - 탭마다 독립적, 도메인과 서브도메인 구분
  - 수명: 탭이 켜져있을 때까지
  - 데이터 휘발성: 탭을 종료하면 삭제됨
  - 용량: 도메인/서브도메인당 약 5MB
  - 통신: 서버에 자동 전송되지 않음
- Cookie
  - 저장 범위: 도메인/경로
  - 수명: 설정 가능(Expires, Max-Age)
  - 데이터 휘발성: 세션쿠키는 브라우저 종료 시 삭제. 지속쿠키는 설정된 만료일까지 유지
  - 용량: 쿠키당 약 4KB
  - 특이사항: HttpOnly, Secure 같은 보안 속성 지원. 서버로 자동 전송되므로 민감한 데이터 저장 시 조심해야 함
- 공통
  - 데이터 형식은 문자열만 허용
  - LocalStorage, SessionStorage에 저장된 데이터는 암호화되지 않으며 XSS 공격에 노출될 수 있어 주의해야 함
