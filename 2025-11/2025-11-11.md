# 2025-11-11 TIL

## 오늘 배운 것

- git의 기능들
  - add: 스테이지로 파일 추가
  - commit: 체크포인트 생성
  - push: 로컬 저장소에서 원격 저장소로 변경사항 반영
  - pull: 원격 저장소에서 로컬 저장소로 변경사항 fetch & merge
  - fetch: 원격 저장소로부터 로컬 저장소를 업데이트
  - merge: 두 브랜치의 커밋을 병합
  - clone: 원격 저장소를 로컬 저장소로 복제
- .gitignore로 변경사항 추적 제한
- markdown 문법 및 github profile 설정
