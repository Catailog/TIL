# 2025-12-08 TIL

## 오늘 배운 것

- 자바스크립트 OOP(생성자, 상속, private), ajax(XMLHttpRequest, Promise, fetch, async/await)
- 메서드는 모든 인스턴스가 공유해서 사용하여 메모리를 절약한다. 하지만 메서드 내에 선언되는 변수는 공유하지 않는다.
- async 함수는 promise를 반환한다. 따라서 return 값을 그대로 사용하려고 하면 오류가 난다. 호출 후 then 메서드를 이용해 데이터에 접근해야 한다. 또는 await으로 async 함수를 호출한 후 바로 결과를 받아 사용할 수도 있지만, await을 사용하려면 await 코드를 감싸는 함수에 async 키워드를 붙여야 한다.

```js
// 호출할 async 함수
async function getUser() {
  return { name: "홍길동", age: 20 };
}

// await 없이 호출
const user = getUser();
console.log(
  user.then((result) => {
    console.log(result);
  })
);

// await과 함께 호출
(async function () {
  const user = await getUser();
  console.log(user);
})();
```
