## 오늘 배운 것

- fetch 데이터 렌더링 실습, 설치 권장 프로그램, Fragment
- Fragment를 사용하면 <div>와 같은 별도의 노드를 추가하지 않고 여러 개의 자식을 감싸줄 수 있다. <Fragment>의 형태로 사용해도 되지만, <>와 같이 아무 이름 없이 쓸 수도 있다. key가 필요한 경우 <Fragment>의 형태로 적어야 하고, Fragment를 적을 경우 import가 필요하다.
- NavLink의 end prop을 true로 넘기면 하위 경로는 활성화 조건에서 제외된다.
  - 예: /main/sub 경로에서 main NavLink가 기본적으로 활성화되지만, end={true}를 설정하면 /main 경로일 때만 활성화된다. 혹은 그냥 end라고만 적어줘도 된다.

```tsx
export function NavLinkItem({ to, end, children }: NavLinkItemProps) {
  return (
    <NavLink
      to={to}
      end={end}
      className={({ isActive }) =>
        `text-blue-500 ${isActive ? "font-bold" : ""}`
      }
    >
      {children}
    </NavLink>
  );
}
```

```tsx
<NavLinkItem to="/lecture04" end>Home</NavLinkItem>
<NavLinkItem to="/lecture04/about">About</NavLinkItem>
```
