# 2026-01-27 TIL

## 오늘 배운 것

- Axios, React Hook Form

- Axios interceptor를 사용하면 요청 전에 인증 토큰이나 API 키 등 헤더를 설정할 수 있다. 응답을 받은 뒤에도 실행할 수 있다. 공통 에러 처리, 데이터 변환 등에 사용된다. 마운트 시 use, 언마운트 시 eject로 인터셉터를 적용/제거할 수 있다.
- AxiosInstance를 사용하면 baseURL, timeout, headers 등의 전역 설정을 할 수 있다.

```tsx
const jsonPlaceholderAPI: AxiosInstance = axios.create({
  baseURL: "<https://jsonplaceholder.typicode.com>",
  timeout: 10000,
  headers: {
    "Content-Type": "application/json",
    "X-Custom-Header": "CustomValue",
  },
});
```

- React Hook Form을 사용하면 폼 상태를 간편하게 관리할 수 있다. 다양한 방식으로 쓸 수 있지만 `const { register, handleSubmit, formState: { errors } } = useForm<FormValues>();`의 형태가 가장 기본적인 형태이다. 이 중 요소의 필드를 입력하는 register 함수는 `<input {...register("username", { required: "Username is required" })} />`처럼 전개구문을 통해 props를 전달해주면 된다.
