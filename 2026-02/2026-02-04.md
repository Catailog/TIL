## 오늘 배운 것

- Firebase Authentication, Protected Route

- 권한이 없는 사용자가 특정 페이지에 접근하는 것을 막는 보안 패턴을 보호된 라우트(Protected Route)라 한다. 인증 여부에 따라 목표 페이지로 바로 이동할지, 로그인 페이지로 이동할지 결정하는 조건부 라우팅을 하는 과정이다. 이때 React Router 패키지를 사용하여 Navigate 컴포넌트에 로그인 후 돌아올 페이지를 지정하고 히스토리를 남기지 않도록 하는 속성을 설정할 수 있다.

```tsx
const location = useLocation();
// 로그인하지 않은 경우 로그인 페이지로 리다이렉트
if (!user) {
  // state에 현재 위치를 저장하여 로그인 후 돌아올 수 있게 함
  return <Navigate to="/login" state={{ from: location }} replace />;
}
```
