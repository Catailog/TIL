# 2026-02-09 TIL

## 오늘 배운 것

- shadcn/UI, pnpm

- shadcn/ui는 컴포넌트를 라이브러리 형태로 설치하는 것이 아니라, 코드를 프로젝트로 직접 가져와서 사용한다. 따라서 외부 라이브러리에 의존하지 않고 UI의 모든 부분을 직접 수정할 수 있으며 TailwindCSS와 결합되어 있어 일관되게 디자인 하기에 최적화 되어있다.

- pnpm은 여러 프로젝트에 중복된 모듈을 설치하면서 생기는 문제점들을 개선해준다. 한 곳에 모듈을 설치하고 프로젝트들은 바로가기 연결을 통해 사용하기 때문에 용량 측면에서도 좋고, 매번 다운로드 하지 않기 때문에 속도 측면에서도 좋다.

- TailwindCSS의 주목적은 CSS 유지보수의 최소화에 있는데 @layer components를 통해 클래스를 추상화 한다면 이는 결국 기존의 순수 CSS로 회귀하는 것이 아닐까?
